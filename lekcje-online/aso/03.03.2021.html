Temat: Przygotowanie do pracy serwera DHCP.

Protokół i usługa DHCP (Dynamic Host Configuration Protocol) służy do automatycznej (dynamicznej) konfiguracji hostów w lokalnej sieci komputerowej przez nadanie im:

  • - adresu IP
  • - maski podsieci
  • - adresów serwerów DNS
  • - adresu bramy

Pojęcia dotyczące DHCP:

  • - brama (z ang. gateway) - jest to urządzenie, przez które komputery w sieci lokalnej mogą wymieniać dane z siecią WAN
  • - maska podsieci - liczba, za pomocą której można wyodbrębnić w adresie IP adres sieci LAN oraz adres komputera użytkownika tej sieci
  • - wykluczenie adresów IP - adresy IP, które nie powinny być przydzielane dynamicznie
  • - zastrzeżenie adresów IP - adresy IP, które powinny być przypisane na stałe do danego komputera sieci LAN
  • - czas dzierżawy - czas, na jaki będzie dzierżawiony adres IP
